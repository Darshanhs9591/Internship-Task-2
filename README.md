# Student Record Management System (CLI)
**Task 2 – Java Developer Internship**

## Project Overview
A simple command-line Student Record Management System (CRUD) built in Java. The system allows adding, viewing, updating, and deleting student records stored in an `ArrayList`.

## Features
- Add student (ID, name, marks)
- View all students
- Update student (by ID)
- Delete student (by ID)
- View student by ID
- Input validation and simple error handling

## Files
- `StudentManagementSystem.java` — single-file application containing the `Student` class and CLI logic.
